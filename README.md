## General info
A social media  web application similar to Facebook.


	
## Technologies
Project is created with:
* React
* Node.js
* Express
* Sequelize
* Bootstrap CSS
* Mysql
* AWS S3
* AWS RDS

	
## Setup
To run this project, install it locally using npm:

```
$ cd server
$ npm install
$ npm start
$ cd client
$ npm install
$ npm start
```
